# Creates an expanded dataset for use with [`SEQuential()`](https://causalinference.github.io/SEQTaRget/dev/reference/SEQuential.md)

Creates an expanded dataset for use with
[`SEQuential()`](https://causalinference.github.io/SEQTaRget/dev/reference/SEQuential.md)

## Usage

``` r
SEQexpand(params)
```

## Arguments

- params:

  SEQparams object built in the SEQuential function
